# Інструкція для персоналу «Зірочка»

1. **Запуск POS**  
   - Увімкніть планшет → відкрийте застосунок **Zirochka POS**.  
   - Переконайтеся, що Wi‑Fi увімкнено.

2. **Прийом замовлення**  
   - Перейдіть на вкладку **Меню**, оберіть категорію, натисніть на страву, щоб додати у кошик.  
   - На вкладці **Кошик** перевірте позиції, натисніть **Оформити** для створення замовлення.  
   - Замовлення зберігається офлайн і синхронізується при появі інтернету.

3. **Перегляд історії**  
   - Вкладка **Замовлення** показує останні чеки та суми.

4. **Оновлення меню**  
   - Вкладка **Адмін** → **Перечитати меню з assets**.  
   - Використовуйте після оновлення файлу `menu_zirochka.json`.

5. **Telegram-сповіщення**  
   - Якщо налаштовано токен і chat_id, кожне замовлення дублюється у Telegram.

6. **Поради по роботі з планшетом Samsung**  
   - Вимкніть оптимізацію батареї для Zirochka POS та MacroDroid.  
   - Закріпіть застосунок у пам’яті (App pinning), щоб уникнути випадкового закриття.

7. **Проблеми**  
   - Немає інтернету: замовлення залишаються в базі, синхронізуються пізніше.  
   - Макрос не відповідає: перевірте доступ до сповіщень та URL Apps Script.  
   - Дані не летять у Sheets: оновіть URL у BuildConfig/gradle.properties та перепублікуйте.
