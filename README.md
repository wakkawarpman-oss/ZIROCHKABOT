# ZIROCHKABOT

Налаштування MacroDroid для ресторану «Зірочка» (Samsung-планшет). Це інструкція для створення макросу ввічливих автоматичних відповідей в Instagram із урахуванням часу доби, тону повідомлення та етикетних фраз.

## 1. Створення макросу
1. Відкрийте **MacroDroid** → **+ Додати макрос**.
2. Назва: **«Зірочка – ввічлива комунікація»**.

## 2. Тригер
- **Сповіщення → Отримано сповіщення → Instagram** (без фільтра тексту).

## 3. Отримати дані зі сповіщення
1. **Змінні → Отримати частини тексту сповіщення**
   - Заголовок → `username`
   - Текст → `message`
2. **Змінні → Отримати поточний час** → `current_hour` (0–23).

## 4. Привітання за часом (`greeting`)
- Якщо `%current_hour%` ≥ 8 і `< 12` → `greeting = "Доброго ранку"`
- Інакше якщо `%current_hour%` ≥ 12 і `< 17` → `greeting = "Доброго дня"`
- Інакше якщо `%current_hour%` ≥ 17 і `< 23` → `greeting = "Доброго вечора"`
- Інакше якщо `%current_hour%` == 23 АБО `%current_hour%` < 8 → `greeting = "Доброї ночі"`

## 5. Визначення тону (`tone`)
- Якщо `%message%` містить "добрий день" / "будь ласка" / "дякую" → `tone = "formal"`
- Інакше якщо містить "привіт" / "хай" / "бро" → `tone = "friendly"`
- Інакше якщо містить "чому" / "довго" / "де" / "коли" → `tone = "apologetic"`
- Інакше → `tone = "neutral"`

## 6. Формування відповіді (`final_reply`)
Створіть гілки для кожного тону та дію **Призначити змінну** зі своїм текстом.

### formal
```
%greeting%, %username%! 
Дякуємо, що обрали «Зірочку». 
Ми отримали ваше повідомлення: "%message%". 
Бронювання підтверджено! Чекаємо на вас у гості. 
Стежте за нами в Instagram @zirochka.kyiv 🌟
```

### friendly
```
Привіт, %username%! 😊 
Дякуємо, що написали. 
Ваше повідомлення "%message%" – отримано. 
Будемо раді бачити в «Зірочці»! 
Усі новини в Instagram @zirochka.kyiv 🌟
```

### apologetic
```
%greeting%, %username%! 
Перепрошуємо за будь-які незручності. 🙏 
Ми отримали ваше повідомлення: "%message%". 
Уже все перевіряємо. Дякуємо за терпіння! 
Ми завжди на зв'язку в Instagram @zirochka.kyiv
```

### neutral
```
%greeting%, %username%! 
Дякуємо за повідомлення. 
Найближчим часом ми з вами зв'яжемося. 
А поки – заходьте на наш Instagram @zirochka.kyiv, там завжди настрій ✨
```

## 7. HTTP POST до Google Sheets
**Мережа → HTTP запит**  
- Метод: POST  
- URL: ваш Apps Script (наприклад `https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec`; замініть `YOUR_SCRIPT_ID` на ID свого вебдеплою: Apps Script → Deploy/Manage deployments → **Web app** URL; спершу опублікуйте як Web app, структура URL може відрізнятися для інших типів деплою)  
- Тіло: `{"username": "%username%", "message": "%message%"}`  
- Content-Type: `application/json`

## 8. Надсилання відповіді в Instagram
1. **Екран → Відкрити додаток → Instagram**
2. **Екран → Затримка 2 с**
3. **Екран → Вставити текст** → `%final_reply%`
4. **Екран → Натиснути Enter** (або торкання кнопки «Надіслати» з підібраними координатами).
> Переконайтеся, що MacroDroid має дозвіл на дії на екрані (Android → Спеціальні можливості → MacroDroid); після надання дозволу перезапустіть макрос/додаток, щоб він почав працювати.

## 9. Збереження і перевірка
1. Збережіть макрос (зелений перемикач).
2. Відправте тестові повідомлення у різний час і з різними тонами, перевірте:
   - Привітання відповідає часу.
   - Обрана правильна тональність і шаблон.
   - Відповідь надсилається в Instagram.
   - Дані доходять у Google Sheets.

## 10. Майбутнє розширення
За потреби можна додати режим для роботи офіціантів/ POS та розширити гілки макросу.
