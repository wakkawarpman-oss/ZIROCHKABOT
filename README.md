## Оновлення для Samsung Galaxy Tab (One UI)

### Спеціальні налаштування для Samsung (One UI)

Оскільки Samsung має власну оболонку, яка може "вбивати" фонові додатки, виконайте додаткові кроки:

1. **Вимкнення оптимізації батареї для MacroDroid (Samsung-спосіб):**
   - Після встановлення MacroDroid зайдіть у **Налаштування → Акумулятор і догляд за пристроєм → Акумулятор → Фонове використання**.
   - Знайдіть MacroDroid у списку і виберіть **"Не оптимізувати"** (або переведіть перемикач у вимкнене положення, якщо є опція "Дозволити фонову активність").

2. **Закріплення додатка в пам'яті (не вивантажувати):**
   - Відкрийте список останніх додатків (квадратна кнопка навігації).
   - Знайдіть MacroDroid, натисніть на його іконку зверху і виберіть **"Заблокувати цей додаток"** (або "Тримати відкритим").

3. **Режим "Не турбувати" – вимкнути (або налаштувати):**
   - Переконайтеся, що на планшеті не активовано режим "Не турбувати", який блокує сповіщення. Або додайте Instagram у винятки цього режиму.

4. **Екран завжди увімкнено:**
   - Зайдіть у **Налаштування → Дисплей → Тайм-аут екрана** → оберіть **10 хвилин** (максимум).
   - За потреби увімкніть **Налаштування для розробників → "Не вимикати екран під час заряджання"**.

5. **Always On Display (AOD):**
   - Якщо екран постійно ввімкнено, AOD не потрібен. За бажанням вимкніть для економії ресурсів.

6. **Фінальна перевірка для Samsung:**
   - Надішліть повідомлення в Instagram і через 30 секунд перевірте таблицю.
   - Через 1 годину повторіть тест. Якщо друге не спрацювало, поверніться до пункту про закріплення MacroDroid.

Код Google Apps Script (`Code.gs`) і шаблон MacroDroid (`instagram_to_sheets.mdr`) універсальні та не залежать від моделі.
