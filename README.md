# Instagram Direct → Google Sheets міст для бронювань

Цей проєкт дозволяє автоматично записувати повідомлення з Instagram Direct у Google Таблицю, **без використання офіційного Meta API** та без щомісячної плати. Все працює на звичайному Android-телефоні, який постійно знаходиться в ресторані.

## Як це працює

1. Клієнт надсилає повідомлення в Instagram Direct ресторану.
2. Android-телефон отримує сповіщення.
3. Додаток **MacroDroid** перехоплює текст сповіщення.
4. MacroDroid відправляє HTTP-запит на **Google Apps Script**.
5. Скрипт додає новий рядок у **Google Sheets** (дата, ім'я користувача, текст повідомлення).

## Вимоги

- Будь-який Android-телефон (Android 6+; перевірте актуальні вимоги MacroDroid) із підтримкою Wi-Fi та підзарядкою.
- Обліковий запис Google (для створення таблиці та Apps Script).
- Акаунт Instagram ресторану.

## Інструкція з налаштування (10 хвилин)

### Крок 1. Підготовка Google Таблиці та Apps Script

1. **Створіть нову Google Таблицю** (наприклад, "Бронювання ресторану").
2. Перейменуйте перший лист на **Бронювання** (важливо: назва має співпадати з тією, що в коді).
3. В меню виберіть **Розширення → Apps Script**.
4. Видаліть весь код у редакторі та вставте код із файлу [`Code.gs`](Code.gs) цього репозиторію.
5. Натисніть **Зберегти** (іконка дискети) і назвіть проєкт, наприклад, `InstagramBridge`.
6. Натисніть **Розгорнути → Новий варіант розгортання**.
   - **Тип:** Веб-додаток
   - **Виконувати як:** Я (ваш обліковий запис)
   - **Доступ:** Усі (або "Усі, хто має посилання")
7. Натисніть **Розгорнути**.
8. **Надайте дозволи** – оберіть свій акаунт, натисніть "Додаткові параметри" → "Перейти на сторінку ... (небезпечно)" → "Дозволити".
9. **Скопіюйте URL веб-додатку** (він знадобиться далі).

### Крок 2. Налаштування Android-телефона

#### 2.1. Встановіть необхідні додатки
- [Instagram](https://play.google.com/store/apps/details?id=com.instagram.android)
- [MacroDroid](https://play.google.com/store/apps/details?id=com.arlosoft.macrodroid) (безкоштовна версія)

#### 2.2. Увійдіть в Instagram під акаунтом ресторану

#### 2.3. Вимкніть блокування екрана
- Зайдіть у **Налаштування Android → Дисплей → Тайм-аут екрана** → виберіть максимальний час (або режим розробника для "Ніколи не вимикати").
- Рекомендується також вимкнути автоматичне блокування через **Блокування екрана → Ніколи**.

#### 2.4. Імпортуйте готовий макрос MacroDroid (опційно, якщо файл уже є в релізі)
Якщо в релізах немає готового файлу або у вас немає доступу до релізів, пропустіть цей підрозділ і перейдіть одразу до розділу "Як створити макрос вручну" нижче.
1. Завантажте файл **`instagram_to_sheets.mdr`** з [останнього релізу](https://github.com/wakkawarpman-oss/ZIROCHKABOT/releases/latest) цього репозиторію.
2. Відкрийте MacroDroid → натисніть меню (три смужки) → **Імпортувати макрос** → виберіть завантажений файл.
3. Підтвердьте імпорт.

#### 2.5. Налаштуйте URL у макросі
- Після імпорту макрос з'явиться в списку (назва "Instagram → Sheets").
- Натисніть на нього, щоб відкрити редагування.
- Знайдіть дію **HTTP запит** (вона буде виділена червоним, бо URL не заповнений).
- Вставте скопійований раніше URL веб-додатку Apps Script.
- Натисніть **Зберегти** (галочка вгорі).

#### 2.6. Увімкніть макрос
- Переведіть перемикач біля макросу в активний стан (зелений).

#### 2.7. Надайте дозволи MacroDroid
- При першому запуску Android попросить надати дозволи для MacroDroid: доступ до сповіщень, до Інтернету тощо. Надайте всі.
- **Важливо!** Зайдіть у **Налаштування Android → Батарея → Оптимізація батареї** → знайдіть MacroDroid і виберіть **Не оптимізувати**. Це запобігає "засинанню" додатка.

#### 2.8. Перевірте роботу
- Надішліть тестове повідомлення в Instagram Direct на акаунт ресторану (з іншого акаунта).
- За кілька секунд перевірте Google Таблицю – у ній має з'явитися новий рядок із текстом повідомлення.

## Як створити макрос вручну (якщо не працює імпорт)

Якщо з якихось причин імпорт `.mdr` не працює, ви можете створити макрос вручну:

1. У MacroDroid натисніть **+ Додати макрос**.
2. **Тригер:** *Сповіщення* → *Отримано сповіщення* → оберіть **Instagram**.
3. **Дія:** *Змінні* → *Отримати частини тексту сповіщення* (налаштуйте, щоб отримати текст повідомлення та ім'я користувача). Або простіше: використайте системну змінну `[notification_text]`.
4. **Дія:** *Мережа* → *HTTP запит*:
   - **Метод:** POST
   - **URL:** (вставте URL Apps Script)
   - **Тіло запиту (Body):** `{"username": "[notification_title]", "message": "[notification_text]"}`
   - **Тип вмісту:** application/json
5. Збережіть макрос.

## Можливі проблеми та їх вирішення

| Проблема | Рішення |
|----------|---------|
| Нічого не записується в таблицю | Перевірте, чи доходить сповіщення до телефона (увімкніть звук). Переконайтеся, що макрос активний. Перевірте URL у HTTP-запиті. |
| MacroDroid "засинає" | Вимкніть оптимізацію батареї для MacroDroid (див. крок 2.7). |
| Помилка CORS у Apps Script | У Apps Script не потрібно налаштовувати CORS – він працює як веб-додаток. |
| Instagram не показує сповіщення | Перевірте налаштування сповіщень у самому Instagram: Профіль → (три смужки) → Налаштування → Сповіщення → Повідомлення → увімкніть. |
