# Зірочка · Instagram-бот для бронювання

Це набір файлів для автообробки повідомлень в Instagram ресторану «Зірочка». Використовуйте Google Apps Script та MacroDroid на планшеті Samsung, щоб приймати бронювання та надсилати шаблонні відповіді з облікового запису `@zirochka.kyiv`.

## Що входить
- `Code.gs` – скрипт Google Apps Script для запису в Google Sheets.
- `zirochka_instagram_bot.mdr` – макрос MacroDroid (URL потрібно вставити самостійно).
- `STAFF_GUIDE.md` – проста інструкція для персоналу.

## Підготовка середовища
1. **Планшет Samsung**: підключіть Wi‑Fi, зарядку, увійдіть в Instagram під `@zirochka.kyiv`.
2. **Залишити в пам’яті**: відкрийте «Налаштування → Обслуговування пристрою → Пам’ять», додайте MacroDroid та Instagram до списку «Ніколи не сплячі програми».
3. **Вимкнути оптимізацію**: у «Налаштування → Батарея та пристрої → Батарея → Обмеження фонової активності» дозволити роботу MacroDroid і Instagram без обмежень.

## Налаштування Apps Script
1. Створіть копію Google Sheets з листом **«Бронювання»**.
2. Додайте файл `Code.gs` у проєкт Apps Script цієї таблиці.
3. Розгорніть вебдодаток як «Anyone with the link» і скопіюйте URL.
4. Замініть плейсхолдер **INSERT_YOUR_APPS_SCRIPT_URL_HERE** (тобто «вставити ваш URL») під час налаштування MacroDroid; у цьому README він наведений лише як приклад і не змінюється.
5. Додайте перевірку секретного токена в заголовку запиту, щоб обмежити доступ до скрипту (рекомендовано як обов’язковий крок безпеки).

## Налаштування MacroDroid
1. Встановіть MacroDroid, імпортуйте файл `zirochka_instagram_bot.mdr`.
2. Відкрийте дію HTTP-запиту та замініть поле URL на ваш розгорнутий Apps Script (**вставити ваш URL**).
3. Збережіть макрос та увімкніть його.
