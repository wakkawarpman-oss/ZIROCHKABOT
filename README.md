# Instagram → Google Sheets міст для ресторану «Зірочка» + автовідповіді ✨

Цей проєкт автоматично записує повідомлення з Instagram Direct у Google Таблицю **та надсилає ввічливу відповідь клієнту**. Ідеально для ресторану **«Зірочка»**, який хоче обійти Meta Business і не платити за підписки.

## Наш Instagram
👉 [@zirochka.kyiv](https://www.instagram.com/zirochka.kyiv)

## Як це працює

1. Гість надсилає повідомлення в Instagram **«Зірочки»**.
2. Samsung-планшет у залі отримує сповіщення.
3. **MacroDroid** перехоплює текст і аналізує його тон.
4. Залежно від тону обирається один із трьох фірмових шаблонів відповіді.
5. Відповідь автоматично надсилається в Instagram.
6. Дані бронювання додаються в Google Sheets (менеджер бачить їх онлайн).

Схема: Instagram → Samsung «Зірочка» → MacroDroid → Google Sheets + авто-відповідь.

## Вимоги

- Будь-який Samsung-планшет (або Android 6+) із Wi-Fi та зарядкою.
- Обліковий запис Google.
- Акаунт Instagram [@zirochka.kyiv](https://www.instagram.com/zirochka.kyiv) (уже є).
- Встановлені додатки: [Instagram](https://play.google.com/store/apps/details?id=com.instagram.android), [MacroDroid](https://play.google.com/store/apps/details?id=com.arlosoft.macrodroid).

## Швидке налаштування (10 хвилин)

### Крок 1. Google Таблиця та Apps Script

1. Створіть нову Google Таблицю **«Бронювання Зірочка»**.
2. Перейменуйте перший лист на **Бронювання**.
3. В меню виберіть **Розширення → Apps Script**.
4. Вставте код із файлу [`Code.gs`](Code.gs) цього репозиторію.
5. Натисніть **Зберегти** → назвіть проєкт **«ZirochkaBridge»**.
6. **Розгорнути → Новий варіант розгортання**:
   - **Тип:** Веб-додаток
   - **Виконувати як:** Я
   - **Доступ:** Усі
7. Натисніть **Розгорнути**, надайте дозволи.
8. **Скопіюйте URL веб-додатку** (він знадобиться в MacroDroid).

### Крок 2. Імпорт фірмового макросу MacroDroid

1. Завантажте файл **`zirochka_instagram_bot.mdr`** з [останнього релізу](../../releases/latest).
2. Відкрийте MacroDroid → меню → **Імпортувати макрос** → виберіть файл.
3. Після імпорту макрос з'явиться з назвою **«Зірочка → Sheets + Reply»**.
   - *Якщо реліз недоступний або файл не завантажується, зберіть макрос вручну за інструкцією нижче.*

### Якщо файл `.mdr` недоступний — зберіть макрос вручну

1. Відкрийте MacroDroid → **+ Додати макрос** → назвіть **«Зірочка → Sheets + Reply»**.
2. **Тригер:** Сповіщення → Отримано сповіщення → виберіть **Instagram**.
3. **Дії:**
   - *Змінні → Призначити значення змінної:* `restaurant_name` = "Зірочка", `restaurant_instagram` = "@zirochka.kyiv", `greeting_formal` = "Доброго вечора", `greeting_friendly` = "Привіт", `apology_text` = "Перепрошуємо".
   - *Змінні → Отримати частини тексту сповіщення:* заголовок → `username`, текст → `message`.
   - *Якщо / Інакше:*  
     1) Якщо `%message%` містить "добрий день"/"будь ласка"/"дякую" → `chosen_reply` = текст для **офіційного** тону (див. таблицю «Шаблони відповідей «Зірочки»» нижче).  
     2) Інакше якщо містить "привіт"/"хай"/"бро" → `chosen_reply` = текст для **дружнього** тону (див. таблицю).  
     3) Інакше якщо містить "чому"/"довго"/"де"/"коли" → `chosen_reply` = текст для **вибачливого** тону (див. таблицю).  
     4) Інакше → `chosen_reply` = текст для **нейтрального** тону (див. таблицю).
   - *Мережа → HTTP запит:* Метод POST, URL — ваш Apps Script, тіло `{"username": "%username%", "message": "%message%"}`, тип вмісту `application/json`.
   - *Екран:* Відкрити Instagram → Затримка 2 с → Вставити текст `%chosen_reply%` → Надіслати (дія MacroDroid **Натиснути клавішу Enter**; якщо не спрацьовує, додайте дію **Натиснути на екран** з координатами кнопки «Send» на вашому пристрої; координати можна вибрати у вбудованому пікері MacroDroid або увімкнути показ дотиків/координат у налаштуваннях розробника Android).
   
   Тексти шаблонів зберігайте в таблиці «Шаблони відповідей «Зірочки»» нижче — вона є єдиним джерелом правди для макросу.
4. Збережіть макрос.

### Крок 3. Усе вже налаштовано під «Зірочку»!

У макросі за вас уже заповнено:

| Змінна | Значення |
|--------|----------|
| `restaurant_name` | «Зірочка» |
| `restaurant_instagram` | @zirochka.kyiv |
| `greeting_formal` | Доброго вечора |
| `greeting_friendly` | Привіт |
| `apology_text` | Перепрошуємо |

Вам залишається лише:
- **Вставити URL вашого Apps Script** у дію **HTTP запит** (вона буде червоною).
- Перевірити, чи правильно підхоплюються змінні (не змінюйте їх, якщо не потрібно).

### Крок 4. Увімкнення макросу та дозволи

- Активуйте макрос (зелений перемикач).
- Надайте MacroDroid доступ до сповіщень, Інтернету.
- **Для Samsung обов'язково:**
  - Відкрийте список останніх додатків → знайдіть MacroDroid → натисніть на іконку → **Заблокувати додаток**.
  - Зайдіть у **Налаштування → Акумулятор → Фонове використання** → MacroDroid → **Не оптимізувати**.

### Крок 5. Тестування

Надішліть тестове повідомлення з іншого акаунта в [@zirochka.kyiv](https://www.instagram.com/zirochka.kyiv). Протягом 10-20 секунд:
- У Google Sheets має з'явитися новий рядок.
- У Instagram має прийти автоматична відповідь у стилі «Зірочки».

## Шаблони відповідей «Зірочки» ✨

Макрос автоматично обирає тон залежно від повідомлення гостя:

| Тон | Коли використовується | Текст відповіді |
|-----|----------------------|-----------------|
| **Офіційний** | Повідомлення містить "добрий день", "будь ласка", "дякую" | «Доброго вечора, %username%! ✨ Дякуємо, що обрали «Зірочку». Ми отримали ваше повідомлення: "%message%". Запит на бронювання прийнято, підтвердимо час найближчим часом. Стежте за нами в Instagram @zirochka.kyiv 🌟» |
| **Дружній** | Повідомлення містить "привіт", "хай", "бро" | «Привіт, %username%! 😊 Дякуємо, що написали. Ваше повідомлення "%message%" – отримано. Будемо раді бачити в «Зірочці»! Усі новини в Instagram @zirochka.kyiv 🌟» |
| **Вибачливий** | Повідомлення містить "чому", "довго", "де", "коли" | «%apology_text%, %username%! 🙏 Ми отримали ваше повідомлення: "%message%". Уже все перевіряємо. Дякуємо за терпіння! Ми завжди на зв'язку в Instagram @zirochka.kyiv» |
| **Нейтральний** | Інші повідомлення | «%greeting_formal%, %username%! Дякуємо за повідомлення. Найближчим часом ми з вами зв'яжемося. А поки – заходьте на наш Instagram @zirochka.kyiv, там завжди настрій ✨» |

## Як змінити текст відповідей (якщо треба)

Усі тексти зберігаються у змінних на початку макросу:
- `reply_formal`
- `reply_friendly`
- `reply_apologetic`
- `reply_neutral`

Просто відредагуйте їх у MacroDroid – і нові відповіді почнуть діяти одразу.

## Вирішення проблем

| Симптом | Ймовірне рішення |
|---------|------------------|
| Відповідь не надсилається | Перевірте, чи має MacroDroid дозвіл на виконання дій на екрані (у налаштуваннях Android -> Спеціальні можливості -> MacroDroid -> увімкнути). |
| Відповідь надсилається, але не та | Перевірте, чи правильно заповнені змінні з текстами (вони мають містити `%username%` тощо). |
| Планшет "засинає" | Виконайте кроки для Samsung (закріплення в пам'яті, вимкнення оптимізації батареї). |
