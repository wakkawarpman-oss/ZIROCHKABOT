# ZIROCHKABOT Canva Video Kit

Цей пакет допоможе швидко зібрати відеоінструкцію в Canva без ручного переписування тексту.

## Що всередині

- `slides_bulk_create.csv` — дані для Canva Bulk Create (15 сцен)
- `voiceover_script_uk.md` — готовий текст для озвучки по сценах
- `timing_plan.md` — тривалості сцен і контрольний чекліст
- `slides_bulk_create_short.csv` — коротка версія (7 сцен, ~5:50)
- `voiceover_script_uk_short.md` — коротка озвучка
- `timing_plan_short.md` — короткий таймінг і чекліст

## Як зібрати відео в Canva (швидкий флоу)

1. Створіть дизайн `Video (16:9)`.
2. Відкрийте `Apps` -> `Bulk Create` -> `Upload CSV` і завантажте `slides_bulk_create.csv`.
3. Створіть базовий шаблон слайда з полями (title, subtitle, line1...line6, footer).
4. Зв'яжіть текстові поля з колонками CSV через Bulk Create.
5. Згенеруйте 15 сторінок.
6. Для кожної сцени додайте фон/ілюстрацію через `Magic Media` з колонки `bg_prompt_en`.
7. Додайте анімацію `Fade` (або `Плавна поява`) для заголовка і списків.
8. Встановіть тривалість сторінок за `duration_sec` (див. `timing_plan.md`).
9. Додайте озвучку за `voiceover_script_uk.md`.
10. Експорт: `MP4 (1080p)`.

## Коротка версія 5-6 хв

Якщо потрібна компактна інструкція, використовуйте:

- `slides_bulk_create_short.csv`
- `voiceover_script_uk_short.md`
- `timing_plan_short.md`

Цей набір стискає матеріал до ~5:50 і об'єднує кроки в 7 сцен.

## Візуальні стандарти

- Шрифти: `Montserrat Bold` (заголовки), `Roboto` (текст).
- Кольори:
  - фон: `#F9F3E7` або білий,
  - акцент: `#D4AF37`,
  - текст: `#333333`.
- Малий логотип/зірка в кутку кожної сцени.
